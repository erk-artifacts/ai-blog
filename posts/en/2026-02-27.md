## What is Meta-prompting?

> Meta-prompting is, in a nutshell, **"a prompt for creating prompts," or "high-level instructions for controlling an AI's behavior and thought processes."** While ordinary prompts ask an AI for "answers," meta-prompting provides an AI with structure and rules about "how to think and how to construct responses."

The AI explains it this way, but

**essentially, it's about having the AI write prompts for you!**

For example, if you give an instruction like:

```
Generate a prompt for creating materials.
```

The AI will produce something like:

```
You are an excellent 【role (e.g., management consultant / marketer / sales representative)】.
Create 【type of material (e.g., new business proposal / sales presentation structure / operational improvement proposal)】 according to the following requirements.

# Purpose
・【What you want to achieve through this material (e.g., get your boss to approve the budget / convey the appeal of a new service to customers)】

# Target (readers/audience)
・【Who this material is for (e.g., management with low IT literacy / existing customer representatives)】

# Elements and preconditions to include
・【Element 1 (e.g., current issue: 〇〇costs are high)】
・【Element 2 (e.g., solution: cost automation through new system implementation)】
・【Element 3 (e.g., expected outcome: 20% cost reduction)】

# Tone & manner
・【Desired writing style (e.g., logical and persuasive tone / simple expressions avoiding technical jargon)】

# Output format
・【Desired format (e.g., provide a 5-page slide structure plan with titles and key messages for each page in bullet points)】
```

There is [research](https://arxiv.org/abs/2503.16789) showing that having LLMs write prompts themselves works **better**, so please take advantage of this technique!
