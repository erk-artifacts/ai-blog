## 元提示词是什么？

> 元提示词（Meta-prompting）简单来说就是**"用于创建提示词的提示词"，或者"为了让AI控制自身行为和思考过程而给出的高阶指令"**。与通常要求AI给出"答案"的提示词不同，元提示词为AI提供了"应该如何思考、如何构建回答"的结构和规则。

虽然AI如上所述进行了解释，但

**简而言之，就是让AI为你编写提示词！**

例如，

```
请为资料创建生成一个提示词。
```

这样指示时，

```
你是一位优秀的【职位（例：经营顾问 / 营销人员 / 销售代表）】。
根据以下要求，创建【资料类型（例：新业务企划书 / 销售演示资料结构 / 业务改进提案书）】。

# 目的
・【通过该资料想要达成的目标（例：获得上司的预算批准 / 向客户传达新服务的魅力）】

# 目标受众（读者·听众）
・【资料的对象（例：IT素养不高的管理层 / 现有客户的负责人）】

# 需要包含的要素·前提条件
・【要素1（例：现状课题：〇〇成本过高）】
・【要素2（例：解决方案：通过引进新系统实现自动化）】
・【要素3（例：预期效果：成本削减20%）】

# 语气和风格
・【期望的文体（例：逻辑严密且有说服力的语气 / 避免专业术语的易懂表达）】

# 输出格式
・【期望的格式（例：请以全5页幻灯片构成方案的形式输出，用要点列表展示各页的标题和应该传达的信息）】
```

这样就能为你生成提示词。

有[研究](https://arxiv.org/abs/2503.16789)表明，让LLM自身编写提示词**效果更好**，请务必利用这一方法！
