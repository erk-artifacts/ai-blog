## メタプロンプトとは？

> メタプロンプト（Meta-prompting）とは、一言で言えば**「プロンプトを作るためのプロンプト」、あるいは「AIに自身の振る舞いや思考プロセスを制御させるための高次元な指示」**のことです。 通常のプロンプトがAIに「答え」を求めるのに対し、メタプロンプトはAIに「どのように考え、どのように回答を構築すべきか」という構造やルールを与えます。

上記のようにAIは説明してくれていますが、

**要は、AIにプロンプトを書かせるということです！**

例えば、

```
資料作成のためのプロンプトを生成してください。
```

などのように指示すると、

```
あなたは優秀な【役割（例：経営コンサルタント / マーケター / 営業担当）】です。
以下の要件に従って、【資料の種類（例：新規事業の企画書 / 営業用プレゼン資料の構成 / 業務改善の提案書）】を作成してください。

# 目的
・【この資料を通じて達成したいこと（例：上司に予算の承認をもらう / 顧客に新サービスの魅力を伝える）】

# ターゲット（読者・聞き手）
・【誰に向けた資料か（例：ITリテラシーが高くない経営層 / 既存顧客の担当者）】

# 盛り込みたい要素・前提条件
・【要素1（例：現状の課題：〇〇のコストが高い）】
・【要素2（例：解決策：新システム導入による自動化）】
・【要素3（例：期待できる効果：コスト20%削減）】

# トーン＆マナー
・【希望する文体（例：論理的で説得力のあるトーン / 専門用語を避けた分かりやすい表現）】

# 出力形式
・【希望する形式（例：全5ページのスライド構成案として、各ページのタイトルと伝えるべきメッセージを箇条書きで出力してください）】
```
のように、プロンプトを生成してくれます。

LLM自身にプロンプトを書かせるほうが**うまくいく**という[研究](https://arxiv.org/abs/2503.16789)がありますので、ぜひご活用くださいませ！
